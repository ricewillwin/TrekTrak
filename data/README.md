# Data

Exported datasets from TrekTrak simulation runs and pipeline deployments.

## Files

| File | Description |
|---|---|
| `sim.csv` | One simulated 24-hour day (96 × 15-min windows) generated by `simulation.py --fast --seed 42` |

## Generating / Refreshing sim.csv

```bash
python simulation/simulation.py --fast --seed 42 --csv data/sim.csv
```

Remove `--seed 42` for a fresh random run. Add `--days N` to generate multiple days.
